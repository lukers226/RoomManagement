2. Update `.env` with your actual Firebase configuration values

3. The following environment variables are required:
   - `FIREBASE_API_KEY`: Your Firebase API key
   - `FIREBASE_APP_ID_ANDROID`: Android app ID

   - `FIREBASE_MESSAGING_SENDER_ID`: Messaging sender ID
   - `FIREBASE_PROJECT_ID`: Firebase project ID
   - `FIREBASE_STORAGE_BUCKET`: Storage bucket URL
   - `ADMIN_PHONE_NUMBERS`: Comma-separated list of admin phone numbers
